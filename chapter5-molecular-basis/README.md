# Molecular Basis of Inheritance

Premium free Class 12 Biology visual chapter website built with React, Vite, Tailwind CSS, Three.js, Mermaid, original schematic visuals, and local JSON content.

## Run locally

```bash
pnpm install
pnpm run dev
```

## Deploy

This chapter is deployed as `/chapter5/` through the root GitHub Pages workflow. No backend, database, paid API, stock assets, or subscription is required.
