# BioForge — Sexual Reproduction in Flowering Plants

A separate premium-feeling, completely free interactive website for Class 12 Biology Chapter 1: **Sexual Reproduction in Flowering Plants**.

It uses original SVG/CSS/Three.js visuals, local JSON content, Mermaid flowcharts, quizzes, flashcards, and a NEET-focused revision dashboard. No backend, paid API, stock images, copied NCERT diagrams, or subscription is required.

## Free stack

- React + Vite
- Tailwind CSS
- Three.js
- Mermaid.js
- Local JSON content
- GitHub Pages deployment

## Run locally

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
pnpm run preview
```

## Deploy free on GitHub Pages

Upload this folder to a public GitHub repository and enable **Settings → Pages → GitHub Actions**.

The public link will look like:

```text
https://your-github-username.github.io/bioforge-flowering-reproduction/
```
